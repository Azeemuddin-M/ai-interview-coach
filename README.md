🎯 AI Interview Coach

An AI-powered interview practice tool that generates questions and evaluates answers using LLM.

## 🚀 Features

- Select topic (Java, Python, JavaScript)
- AI-generated interview questions
- Answer evaluation with score
- Feedback and correct answers

## 🧠 Tech Stack

- Python
- Streamlit
- Groq API (LLaMA 3.1)

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
